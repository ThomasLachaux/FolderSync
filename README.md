# FolderSync

Bash command for the synchronisation of 2 folders with conflicts management

## How to use

```
chmod +x main.sh
mkdir sync{A,B} # Create two folders
echo 'hello !' > syncA/test
./main.sh syncA syncB # Enjoy !
```
